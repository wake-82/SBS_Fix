# SBS Fix

## Overview / 개요

SBS Fix is a free, privately developed freeware tool designed for simple editing of Full SBS videos. SBS Fix는 개인이 제작한 무료 프리웨어로, Full SBS 영상을 간단하게 편집하는 도구입니다. 본 프로그램은 비상업적 용도에 한해 무료로 사용 가능합니다.

## Key Features / 주요 기능

AI-Powered Scene Analysis (TransNetV2): Uses the TransNetV2 AI engine to automatically detect scene transitions for precise editing.
AI 기반(TransNetV2) 자동 장면 분석: AI 엔진을 사용하여 장면 전환 지점을 자동으로 감지합니다.

Simple Editing Functions: Offers the ability to fix each detected section by choosing Left (L), Right (R), or replacing it with another FSBS source.
간편한 편집 기능: 분석된 각 구간별로 왼쪽(L), 오른쪽(R), 또는 제3의 FSBS 영상으로 교체할 수 있습니다.

High-Performance Video I/O via FFmpeg: Leverages FFmpeg to support various codecs (H.264, H.265/HEVC) and hardware acceleration (NVENC) for efficient encoding.
FFmpeg을 이용한 고성능 영상 입출력: 다양한 코덱과 하드웨어 가속(NVENC)을 지원합니다.

Portable Execution: Runs as a standalone application without the need for a separate installation process.
포터블 실행 지원: 별도의 설치 과정 없이 바로 실행 가능합니다.

## How to Run / 실행 방법

Place SBS_Fix.exe and the _internal folder in your preferred directory.
SBS_Fix.exe와 _internal 폴더를 원하는 폴더에 둡니다.

Set up FFmpeg. (Refer to the instructions below)
FFmpeg을 준비합니다. (아래 안내 참고)

Run SBS_Fix.exe.
SBS_Fix.exe를 실행합니다.

Necessary configuration files will be automatically generated upon startup.
프로그램 실행 시 필요한 설정 파일은 자동으로 생성됩니다.
